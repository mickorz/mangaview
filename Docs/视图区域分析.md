# MangaView 视图区域分析

## 整体布局结构

```
+------------------------------------------------------------------+
|                           HEADER (顶部工具栏)                      |
|  +------------+  +-------------------+  +----------------------+  |
|  | 左侧工具区  |  |    书名/书架入口   |  |     右侧工具区        |  |
|  +------------+  +-------------------+  +----------------------+  |
+------------------------------------------------------------------+
|                                              |                    |
|                                              |                    |
|              MAIN CONTENT                    |    DRAWER          |
|              (漫画阅读区)                     |   (缩略图预览)      |
|                                              |                    |
|     +-----------------------------+          |   +------------+   |
|     |                             |          |   |  thumb 1   |   |
|     |      漫画图片显示区域         |          |   +------------+   |
|     |                             |          |   |  thumb 2   |   |
|     +-----------------------------+          |   +------------+   |
|                                              |   |  thumb 3   |   |
|     左侧点击区 | 右侧点击区                    |   +------------+   |
|     (上一页)   | (下一页)                     |       ...          |
|                                              |                    |
+------------------------------------------------------------------+
|  [隐藏的上翻按钮]                                                  |
|                                              |   [可拖拽调整宽度]   |
|  [隐藏的下翻按钮]                                                  |
+------------------------------------------------------------------+
```

## 一、HEADER 顶部工具栏

```
+----------------------------------------------------------------------------+
|  [打开] [刷新] [帮助]    [当前书名 - 点击打开书架]    [上一本] [下一本]      |
|                                                [适应宽度] [全屏] [菜单]      |
+----------------------------------------------------------------------------+
```

### 1.1 左侧工具区

| 按钮 | 图标 | 功能 | 说明 |
|------|------|------|------|
| 打开文件夹 | folder_open | 打开文件选择器 | 选择图片文件夹加载 |
| 刷新 | refresh | 重新加载页面 | 清空所有数据，重新开始 |
| 帮助 | help_outline | 打开 GitHub 页面 | 跳转到项目主页 |

### 1.2 中间书名区

| 元素 | 功能 | 说明 |
|------|------|------|
| 书名按钮 | 点击打开书架视图 | 全屏显示所有漫画书的缩略图卡片 |

**书架视图弹出层结构：**

```
+------------------------------------------------------------------+
|  [搜索框]  [最小页数]  [最大页数]  [重置]  [关闭]                    |
+------------------------------------------------------------------+
|  +----------+  +----------+  +----------+  +----------+            |
|  | [封面图]  |  | [封面图]  |  | [封面图]  |  | [封面图]  |            |
|  |          |  |          |  |          |  |          |            |
|  +----------+  +----------+  +----------+  +----------+            |
|  | 25P 书名1 |  | 30P 书名2 |  | 28P 书名3 |  | 35P 书名4 |            |
|  +----------+  +----------+  +----------+  +----------+            |
|                              ...                                  |
+------------------------------------------------------------------+
```

### 1.3 右侧工具区

| 按钮 | 图标 | 功能 | 快捷键 |
|------|------|------|--------|
| 上一本 | backward-step | 切换到上一本书 | Page Up |
| 下一本 | forward-step | 切换到下一本书 | Page Down |
| 适应宽度 | arrows-left-right | 切换图片宽度模式 | - |
| 全屏 | open_in_full | 切换全屏模式 | - |
| 菜单 | menu | 显示/隐藏缩略图面板 | - |

**特性说明：**
- 工具栏支持 `reveal` 模式，鼠标移到顶部 100px 内自动显示，移开后隐藏
- 宽度小于 1000px 时，菜单按钮隐藏，显示右下角浮动按钮

## 二、MAIN CONTENT 主内容区

```
+--------------------------------------------------+
|  [上翻按钮]                                         |
|                                                  |
|     +------------+  +------------+                |
|     |            |  |            |                |
|     |  左侧区域   |  |  右侧区域   |                |
|     |  (点击上翻)  |  |  (点击下翻)  |                |
|     |            |  |            |                |
|     +------------+  +------------+                |
|                                                  |
|  +--------------------------------------------+  |
|  |                                            |  |
|  |              漫画图片垂直排列                |  |
|  |                                            |  |
|  |  [img1] -> [img2] -> [img3] -> ...        |  |
|  |                                            |  |
|  +--------------------------------------------+  |
|                                                  |
|  [下翻按钮]                                       |
+--------------------------------------------------+
```

### 2.1 功能区域

| 区域 | 位置 | 功能 | 交互 |
|------|------|------|------|
| 上翻按钮 | 右上角 | 滚动到顶部或上一本书 | 点击 |
| 下翻按钮 | 右下角 | 滚动到底部或下一本书 | 点击 |
| 左侧区域 | 左半边 (50%) | 向上翻页/滚动 | 鼠标释放 |
| 右侧区域 | 右半边 (50%) | 向下翻页/滚动 | 鼠标释放 |
| 图片区 | 中央 | 显示漫画图片 | 懒加载，支持拖拽滚动 |

### 2.2 空状态提示

当没有加载任何漫画时：

```
+------------------------------------------+
|                                          |
|                                          |
|        Drag and drop folders here        |
|                                          |
|           请拖拽文件夹到此处               |
|                                          |
|                                          |
+------------------------------------------+
```

### 2.3 图片显示模式

| 模式 | CSS 类 | 效果 |
|------|--------|------|
| 适应宽度 | fit-width | 图片宽度 100% 填充 |
| 原始比例 | 默认 | 图片最大宽度 100%，保持比例 |

## 三、DRAWER 缩略图预览面板

```
+------------------+
|                  |
|   +-----------+  |
|   |  thumb 1  |  |
|   +-----------+  |
|   |  thumb 2  |  |
|   +-----------+  |
|   |  thumb 3  |  |
|   +-----------+  |
|   |  thumb 4  |  |
|   +-----------+  |
|       ...        |
|                  |
+--||--+
   ||
   || 可拖拽调整宽度
```

### 3.1 功能说明

| 功能 | 说明 |
|------|------|
| 缩略图列表 | 垂直排列当前书籍的所有页面缩略图 |
| 滚动同步 | 与主区域滚动位置同步 |
| 悬停提示 | 显示图片文件名 |
| 拖拽滚动 | 支持鼠标拖拽快速滚动 |
| 宽度调整 | 左边缘可拖拽调整面板宽度 |

### 3.2 交互行为

| 操作 | 效果 |
|------|------|
| 鼠标进入主区域 | 缩略图面板可能隐藏（非固定模式） |
| 鼠标进入缩略图区 | 缩略图面板显示，滚动同步方向反转 |
| 点击缩略图 | 主区域滚动到对应位置 |
| 拖拽左边缘 | 调整面板宽度，保存到 localStorage |

## 四、DRAG INPUT 拖拽上传层

```
+------------------------------------------------------------------+
|                                                                  |
|                        半透明黑色遮罩层                            |
|                                                                  |
|                     (拖拽文件时显示)                              |
|                                                                  |
+------------------------------------------------------------------+
```

**特性：**
- z-index: 9999，覆盖所有内容
- 拖拽文件进入页面时自动显示
- 支持多文件和文件夹选择
- 只接受图片类型文件

## 六、键盘快捷键映射

| 按键 | 功能 | 作用区域 |
|------|------|----------|
| Space / Down | 向下滚动 | 主区域 |
| Shift + Space / Up | 向上滚动 | 主区域 |
| F / Right / Enter | 向后翻页 | 主区域 |

| Page Down | 下一本书 | 全局 |

| Ctrl + C | 复制书名列表 | 全局 |

## 七、响应式设计

### 7.1 桌面端 (>1000px)

```
+------------------------------------------+
|              HEADER (完整)                |
+------------------------------------------+
|                           |               |
|      MAIN CONTENT         |    DRAWER     |
|                           |               |
+------------------------------------------+
```

### 7.2 移动端 (<1000px)

```
+------------------------------------------+
|              HEADER (隐藏菜单按钮)         |
+------------------------------------------+
|                                          |
|              MAIN CONTENT                |
|                                          |
|                              [+ 浮动按钮]  |
+------------------------------------------+
```

- 工具栏的菜单按钮隐藏
- 右下角显示浮动按钮用于打开缩略图面板
- 缩略图面板以覆盖层形式显示
